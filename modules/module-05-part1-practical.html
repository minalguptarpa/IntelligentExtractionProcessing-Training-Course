Part 1 : Practical Exercise

⏱ ~20 min📖 Hands-On🎯 Intermediate

🎯 What You Will Do

  • Reuse a Studio workflow to configure a Generative Classifier and Extractor
  • Set up Taxonomy with document types and extraction fields
  • Configure and run the end-to-end generative extraction workflow
  • Validate extracted results using Validation Station

Walkthrough Video

Watch the full practical walkthrough below. Follow along in your own UiPath environment to see the Generative Classifier and Extractor configured step by step.

UiPath Generative Extractor Workflow — Step-by-Step

1
Setup
Reuse the Earlier UiPath Studio Workflow
  1. Use the UiPath Studio Workflow from the "Structured Documents" module.
  2. Open the Main workflow.
2
Data
Configure Document Inputs
  1. Place sample documents in the Data → Input folder.
3
Configuration
Configure the Taxonomy
  1. Open Taxonomy Manager.
  2. Create the required document types:
    • Group: Certificates
    • Category: Forms
    • Document Type: Degree_Cert
  3. Create 3 extraction fields:
    • UniversityName (type: Text)
    • AwardeeName (type: Text)
    • DateofDegree (type: Date)
  1. Save the taxonomy.
4
Classifier
Add a Generative Classifier
  1. Under the Classify Document Scope in the workflow, add a Generative Classifier.
  2. Click Manage Field Details on the Generative Classifier.
  3. For the Degree_Cert document type, provide a natural-language description and Save.
  1. Example: Degree_Cert → "Is this a Degree Certificate?"
  2. Click Configure Classifier and save the below configuration.
5
Extractor
Configure a Generative Extractor
  1. Under the Data Extraction Scope in the workflow, add a Generative Extractor.
  2. Click Manage Field Details on the Generative Extractor.
  1. Set prompts for each field:
    • UniversityName: What is the name of the institute awarding the Degree?
    • AwardeeName: Name of the person to which degree is awarded?
    • DateofDegree: Date of Issue of degree
  2. Save.
  3. Click Configure Extractor. Do the below configuration and Save.
6
Execution
Execute Extraction
  1. Run the workflow.
  2. Observe the output logs and verify:
    • Taxonomy loaded successfully
    • Classification completed
    • Extraction completed
    • Extracted fields are returned
7
Validation
Validate Extraction Results
  1. In the Present Validation Station after the extraction step.
  2. Review extracted values.
  3. Correct any incorrectly extracted fields.
  4. Submit validation results.
