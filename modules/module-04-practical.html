Practical Exercise : Extracting Data from Invoices

⏱ ~30 min📖 Hands-On🎯 Intermediate

🎯 What You Will Do

  • Follow along with the step-by-step walkthrough video for end-to-end invoice extraction
  • Set up a Modern Document Understanding project for semi-structured invoices
  • Annotate, train, publish, and consume the model in a Studio workflow

Walkthrough Video

Watch the full practical walkthrough below. Follow along in your own UiPath environment to build the invoice extraction project step by step.

UiPath Document Understanding (Modern Projects) — Step-by-Step

📋 Overview
This exercise walks you through the full Modern Project lifecycle: Create → Configure → Upload → Annotate → Validate → Measure → Publish → Automate.
1

Create a New Document Understanding Project

  1. Open the UiPath Document Understanding service in Automation Cloud.
  2. Select Create Project.
  3. Give the project the name Training_Modern and click Save.
  4. Add the document type — choose Invoices and click Add.
2

Configure Extraction Fields

  1. Click the 3 dots (⋮) menu and select Document Type Manager.
  2. Click on the Extraction Fields tab.
  3. Keep only the required fields and remove all others.
  4. Add a new extraction field named ShipToPartyPhoneNumber of type String.
3

Upload Sample Documents

  1. Navigate to the Build section.
  2. Upload your sample invoice documents.
  3. Wait for the documents to be processed.
  4. Verify that all uploaded files are available for annotation.
4

Annotate Documents

  1. Open a sample invoice.
  2. Select a field from the Extraction Fields panel.
  3. Highlight the corresponding value on the invoice.
  4. Map the following fields:
    • ShipToPartyPhoneNumber
    • Vendor Name
    • Invoice Number
    • Total Amount
  5. Annotate table columns for line-item extraction if applicable.
  6. Repeat for all required fields across all documents.
5

Validate Annotations

  1. Review all highlighted fields.
  2. Ensure each field is mapped correctly.
  3. Check confidence scores and extraction quality.
  4. Save the annotations.
6

Check the Measure Section

  1. Go to the Metrics tab in the Measure section.
  2. Ensure you have a satisfactory accuracy percentage for each field.
💡 Tip
For high-stakes use cases aim for 90%+ F1. For workflows with human-in-the-loop validation, 75–80% is acceptable.
7

Publish the Extraction Model

  1. Navigate to the Publish tab.
  2. Click Create Project Version, choose document type Invoices, give it a name, and click Create.
  3. Toggle the Deployed button to make the model live.
8

Create a Studio Automation

  1. Open UiPath Studio.
  2. Create a new automation project.
  3. Run the Studio Web workflow and test it.
  4. Observe the execution logs.
  5. Verify extraction results.
  6. Confirm that invoice data is extracted successfully.
