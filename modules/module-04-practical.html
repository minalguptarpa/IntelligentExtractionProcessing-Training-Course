Practical Exercise : Extracting Data from Invoices

⏱ ~30 min📖 Hands-On🎯 Intermediate

🎯 What You Will Do

  • Follow along with the step-by-step walkthrough video for end-to-end invoice extraction
  • Set up a Modern Document Understanding project for semi-structured invoices
  • Annotate, train, publish, and consume the model in a Studio workflow

Walkthrough Video

Watch the full practical walkthrough below. Follow along in your own UiPath environment to build the invoice extraction project step by step.

UiPath Document Understanding (Modern Projects) — Step-by-Step

1. Create a New Document Understanding Project

  1. Open UiPath Document Understanding Service in Automation Cloud.
  2. Select Create Project.
  3. Give name as Training_Modern and click Save.
  4. Add the document type, choose Invoices and click Add.

2. Configure Extraction Fields

  1. Click on 3 dots and click on Document Type Manager.
  2. Click on the Extraction Fields tab. Keep these fields and remove all the rest of the fields.
  3. Add a new Extraction field called ShipToPartyPhoneNumber of type String.

3. Upload Sample Documents

  1. Navigate to the Build section.
  2. Upload sample invoice documents.
  3. Wait for the documents to be processed.
  4. Verify that all uploaded files are available for annotation.

4. Annotate Documents

  1. Open a sample invoice.
  2. Select a field from the Extraction Fields panel.
  3. Highlight the corresponding value on the invoice.
  4. Map:
    • ShipToPartyPhoneNumber
    • Vendor Name
    • Invoice Number
    • Total Amount
  5. Annotate table columns for line-item extraction if applicable.
  6. Repeat for all required fields.

5. Validate Annotations

  1. Review all highlighted fields.
  2. Ensure each field is mapped correctly.
  3. Check confidence scores and extraction quality.
  4. Save the annotations.

6. Check the Measure Section

  1. Go to Metrics.
  2. Ensure you have satisfactory accuracy percentage for each field.

7. Publish the Extraction Model

  1. Navigate to Publish.
  2. Click on Create Project Version, choose the document type as Invoices, give a name and click on Create.
  3. Toggle the deployed button.

8. Create a Studio Automation

  1. Open UiPath Studio.
  2. Create a new automation project.
  3. Run the studio web workflow and test it.
  4. Observe the execution logs.
  5. Verify extraction results.
  6. Confirm that invoice data is extracted successfully.
