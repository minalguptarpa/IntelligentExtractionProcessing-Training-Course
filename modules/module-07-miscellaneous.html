Important Topics: Resources & Documentation

⏱ ~5 minutes📖 Reference🎯 All Levels

🎯 Learning Objectives

  • Find the right official documentation for Document Understanding and IXP
  • Know where to go for deeper learning and troubleshooting
  • Understand the Monitor section and how to track document processing performance
  • Use the Processed Documents tab to search and review extraction results

📊 Monitor

The Monitor section in Document Understanding gives you a centralised view of how your automations are performing — tracking document volumes, extraction accuracy, AI unit consumption, and human validation activity.

📖 Reference
Full documentation: Monitor — UiPath Document Understanding User Guide

🗂️ Main Areas of Monitor

AreaWhat it shows
Project Performance (Preview)Processed document counts, time saved, and extraction duration. Requires Insights to be enabled in Tenant Profile settings.
Processed DocumentsA searchable list of all documents processed via API or UiPath.DocumentUnderstanding.Activities. Documents are retained for 6 months.

📈 Key Metrics Available

  • Estimated time saved — calculated as: processed documents × manual processing time − validation time
  • AI Unit consumption — overview and breakdown by consumer (API vs RPA)
  • Straight-through processing rate — documents processed without human intervention vs. total
  • Validation time per validator — tracks how long human reviewers spend per document
  • Extraction & classification accuracy — field correction trends and classification confusion matrix
  • Top exceptions — most frequently encountered processing errors

⚙️ System Requirements

  • UiPath.DocumentUnderstanding.Activities v2.7.0 or newer
  • UiPath.IntelligentOCR.Activities v6.20 or newer (for IntelligentOCR support)

📤 Schedule Delivery

You can automate dashboard exports on a recurring schedule. Configure:

  • Schedule — name, recurrence, time, and destination
  • Recipients — email addresses to receive the report
  • Format — PDF, CSV, or PNG

📋 Processed Documents

The Processed Documents tab within Monitor lists every document that has been processed through Document Understanding — whether via API or the UiPath.DocumentUnderstanding.Activities package.

🔍 What You Can See

ColumnDescription
File NameName of the processed document — click to open Document Details
Document TypeThe document type it was classified as
ConsumerWhether it was processed via API or an RPA workflow
Modified DateWhen the document was last processed or updated
ValidatorThe human reviewer who validated the extraction, if applicable
AI Units ConsumedNumber of AI units used to process this document

💡 Key Facts

  • Documents are searchable by file name
  • Records are retained for 6 months from the processing date
  • Click any file name to open Document Details — a deeper view covering extraction results, validation actions, and accuracy per field
